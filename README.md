采用一台电脑（服务器）带动其他电脑（客户端）的工作方式
客户端运行：client.exe或client2.exe 任何一个皆可
服务器运行：serverpause.exe 或 serverpause2.exe 任何一个皆可
相比于蓝牙，dlna，airplay等技术
本程序可以：
轻松实现一对多！
超低资源占用！
人耳无法分辨的延时！
兼容windows，macos，linux全平台（等研究java了再补上安卓端）
